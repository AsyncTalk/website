### 📕 Shownotes

React Server Component 是最近的大事。本期蹭一次热点，锐评 RSC 和 Next.js。

做前端的朋友们，这次，必须得学后端了哦。

03:10 Next.js 的 server component 实现大方向很棒，但是小缺点也不少
